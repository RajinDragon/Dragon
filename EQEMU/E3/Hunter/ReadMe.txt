# Elite Hunter (Public Server Version)

MacroQuest Lua-based hunter tool for rare detection, tracking, and warp support.

## Features

- Custom Named Tracking
- Blacklist System
- Warp / Warp Party Support
- Force Scan
- Search Filter
- Separate Popout Windows
- Persistent Save Files
- Draggable Radar Button

## Commands

/hunter  
Toggle main UI window.

## Installation

1. Place `hunter.lua` in your MQ Lua folder.
2. Run in game:

```
/lua run hunter
```

## Files Created Automatically

- hunter_custom_named.lua
- hunter_blacklist.lua

These are saved in your MQ config directory.

## Notes

Designed for public emu servers where core Lua cannot be modified.
