
Party Manager v5 for MacroQuest
===============================

Installation:
1. Extract into your MacroQuest lua folder.
2. Folder structure:
   /lua/party.lua
   /lua/party/buff_ai.lua
   /lua/party/click_ai.lua
   /lua/party/rez_ai.lua
   /lua/party/loot_manager.lua
   /lua/party/radial.lua

Run:
/lua run party

Hotkey for radial menu (example):
/bind alt+r /radial

Features:
- Party commander window
- Radial wheel for quick commands
- Buff/combat spellset switching
- Clicky usage
- Resurrection AI
- Loot manager integration with e9loot
